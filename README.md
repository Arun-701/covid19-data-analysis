# COVID-19 Dashboard

## Project Overview

This project analyzes global COVID-19 data and presents key insights through interactive visualizations.

The objective is to understand the spread of COVID-19, identify trends in cases and deaths, and provide meaningful insights using data analytics techniques.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Google Colab

## Dataset

COVID-19 Global Dataset

The dataset contains:

- Total Cases
- Total Deaths
- New Cases
- New Deaths
- Country-wise Statistics
- Date-wise Trends

## Analysis Performed

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Country-wise Case Analysis
- Country-wise Death Analysis
- Death Rate Analysis
- Daily Cases Trend Analysis
- Comparative Country Analysis

## Key Insights

- USA recorded the highest number of COVID-19 cases.
- India ranked among the most affected countries.
- Death rates varied significantly across countries.
- Daily new cases showed multiple waves during the pandemic.
- A few countries accounted for a large percentage of total deaths.

## Dashboard Features

- Country-wise COVID-19 statistics
- Death rate comparison
- Daily cases trend analysis
- Interactive visualizations
- Data-driven insights

## Project Outcome

This dashboard helps understand the global impact of COVID-19 and provides insights into pandemic trends through data visualization.
